Informatika

Informatika dalam pengembangan web berfokus pada penggunaan bahasa markup seperti HTML (HyperText Markup Language) untuk menyusun struktur dan konten halaman (seperti teks, gambar, dan tautan), yang kemudian diperindah dan diatur tata letaknya menggunakan CSS (Cascading Style Sheets) untuk aspek visual (warna, font, layout). Selain itu, pengembangan web modern juga melibatkan JavaScript untuk menambahkan interaktivitas dan perilaku dinamis, serta berbagai teknologi backend dan database untuk mengelola data dan logika aplikasi, membentuk dasar dari semua situs web dan aplikasi daring yang kita gunakan sehari-hari.
